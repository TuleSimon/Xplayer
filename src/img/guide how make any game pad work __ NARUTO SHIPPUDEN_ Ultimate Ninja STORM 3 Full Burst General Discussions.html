If u have a gaming pad and it is not working or driver will not install then it has maybe a few reseasons the pc cant see what it is or some other driver isue's

Step 1: Go to the icon of my computer
Step 2 : Click with ur right mouse on the icon then click on manage.
Step 3: There is now a box in front of u call Computermanage with a view tabs, Click on devicemanage.
Step 4: Search ur gaming pad.
Step 5: If u found ur pad then right mouse click, Update Driver (1e option)
Step 6: Select Search on ur pc.
Step 7: Choose from a list on ur pc.
Step 8: Choose : Windows Common Controller for Window Class

Then it should work!
